How to get config

Paste your Firebase web config object below (from the Firebase console → Project settings → Your apps):

Shop PIN (4–8 digits) — set this once, then enter it on every device to access the same data:

  1. Go to console.firebase.google.com and click Add project (free).
  2. Once created, click the </> (Web) icon to register a web app — give it any nickname.
  3. Firebase will show a firebaseConfig object. Copy the whole { ... } block.
  4. In the left sidebar go to Build → Firestore Database → Create database. Choose Start in test mode for now (you can lock it down later).
  5. Come back here, paste the config into the box on the Connect tab, set a PIN, and hit Connect.

Test mode allows open read/write for 30 days — fine for a single-shop tool, but for stronger security later, ask me and I'll give you locked-down Firestore rules tied to your PIN.
